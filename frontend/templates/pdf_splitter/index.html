Instrucciones

Esta herramienta permite dividir un PDF escaneado que contiene múltiples documentos de estudiantes.

  • Formatos soportados: PDF
  • Tamaño máximo: 200 MB
  • El PDF debe tener el RUN visible en cada página
  • El sistema detectará automáticamente los RUNs usando OCR
